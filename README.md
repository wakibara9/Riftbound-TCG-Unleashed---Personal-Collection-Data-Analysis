# Riftbound-TCG-Unleashed---Personal-Collection-Data-Analysis
**Overview**
A data analysis project evaluating card rarity pull rates and distribution variance. Features a comparative Tableau dashboard analyzing personal hit rates against baseline box averages across multiple rarity tiers.
**Motivation**
I started this project to answer a simple curiosity: Did I actually beat the odds on my recent card pulls? 
>To find out, I wanted to:
Stop guessing with raw numbers: Seeing a total count doesn't tell me much. I wanted to see my actual hit percentages sitting right next to the official box averages so I could instantly spot the difference.
>Fix the scaling mess: It’s tough to map a massive $50\%$ drop rate on the same screen as a tiny $0.6\%$ pull. Standard charts either cut off the highs or completely hide the lows.
>Make it easy to read: I wanted a clean, functional layout where I could glance at the data and immediately see exactly where I beat the curve and where I under-performed.


**Tools Used**
- Excel (Data Entry and aggregation)
- Tableau (Data Visualization)

**Process**
* Counted and organized cards from my personal collection
* Organized by cards by rarity
* Aggregated and analyzed data within Excel
  -Calculated average percentages based on publicly disclosed pull rates and community crowd-sourced pull rates
    -(https://riftbound.gg/riftbound-card-rarity-guide-pull-rates/)
  -Calculated my personal percentages
* Connected Excel worksheet to Tableau
* Created visualizations of my analysis to communicate insights

**Limitations**
* Sample Size. Relatively small given the amount of product being printed for TCGs.
* Manual data entry. It is likely that I may have made mistakes during the data entry process.
* Product Sourcing. I sourced both of my Riftbound Unleashed boxes from the same store and do not have knowledge on if both boxes were from the same case or not.
* Changes to Pack Structure. With the Unleashed set, a new rarity (Ultimate) was introduced, which greatly changed the distribution of what customers are able to pull from packs compared to previous sets like Origins and Spiritforged.
  -(https://riftbound.leagueoflegends.com/en-us/news/announcements/the-unleashed-overview/)
